# 01타일 (acmicpc 1904) 문제풀이

타일을 왼쪽에서부터 오른쪽으로 이어서 붙인다고 가정합니다. 예을 들어 N = 3일 때, 경우의 수는 3가지 입니다. [1,1,1], [1,00], [00,1]

N = 4일때, 경우의 수는 5가지입니다. [1,1,1,1] [1,1,00] [1,00,1] [00,1,1] [00,00]

마지막에 붙을 숫자가 하나면 "1" 두개면 "00" 이기 때문에 두개남은 타일(i - 2) + 하나남은 타일(i - 1) 성립됩니다. 팩토리얼 풀이법과 비슷합니다.

# 소스코드 Python

#n = int(input())
#예제 입력
n = 4
dp = [0] * 1000001
dp[1] = 1
dp[2] = 2
for i in range(3, n + 1):
    dp[i] = (dp[i - 2] + dp[i - 1]) % 15746
print(dp[n])
